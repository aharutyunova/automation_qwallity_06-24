## General Overview

This Python program performs various actions related to managing courses. It includes functionalities to add, update, get and delete courses.

### Program Structure

- `Main.py`: Main Python file for executing the program.

## Code Structure

Here's a breakdown of the key components:

Configuration File (config_file.py): Stores configuration parameters such as usernames and passwords required for authentication.
Endpoints File (endpoints_file.py): Contains the URLs for various API endpoints utilized by the program.
Main Script (main.py): Serves as the main entry point for executing the program. It orchestrates the different functionalities like adding, updating, and deleting courses by making requests to the respective API endpoints.

## Prerequisites

Before running the program, ensure that you have the following dependencies installed:

- `requests`
- `logging`
- `re`
- `json`

## Usage

To start the program, run `Main.py`.
